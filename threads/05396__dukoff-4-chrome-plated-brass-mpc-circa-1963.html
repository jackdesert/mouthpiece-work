You can use a facing curve in the Files - Tenor Sax area.  You will 
need MS Excel or a compatible program.  There is a spreadsheet labled 
Tenor Link Facing Curve Table.  Use the column for a Link 7 = .100".   
Just plug in your own set of feelers in the far left column.  The 
glass gage numbers will update automatically.

I might be inclined to go a little longer with the facing length than 
46.  But the one there aint bad.
