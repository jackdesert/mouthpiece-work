SUBJECT: HR Florida Otto Link Facing Chart
Has anyone got a facing chart for a HR "Florida" Otto Link (4* to 5* 
is what I'm looking for)? Since I don't know which feelers are 
supposed to be used for each of the lengths given, it's probably a 
good idea to sa which ones are at which lengths, just to make sure I 
don't misunderstand things.

Alternatively, if anyone has a facing for Getz's mpc's from 
the "Bossa" period (of course, the real facing would be more useful 
than a contrived one, but who knows if Joe's old boss actually had 
the real facing or not???), that would be a great help. This is for 
the Rico Royal mpc that I said about earlier (obviously I'm going to 
grind out the baffle and chamber to be more like a Link's chamber).

Thanks in advance.

Danny
FROM: kwbradbury (Keith Bradbury)
SUBJECT: Re: HR Florida Otto Link Facing Chart
> Alternatively, if anyone has a facing for Getz's mpc's... 

From "Joe" on SOTW (again we can not verify this info):

Stan Getz 
Metal Link STM 
87 tip 
27-20-13-10-6 1/2 (in mm.) Note: Most glass gauges are marked in 1/2 
mm. 
Rico #3 reeds 

Feelers .0015, .010, .024, .034, .050 

Getz did play on a lot of different pieces, but this was his favorite 
facing on metal Links as he had several Links done to this facing.
