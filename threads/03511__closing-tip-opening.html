FROM: kwbradbury (Keith Bradbury)
SUBJECT: Re: Closing Tip Opening
No, you have plenty of material in the table to close it down.  The hard
part is keeping the table flat.  You can bias you table cutting as if
angling it, but it is not as important as keeping th table flat.  If you
lean too hard trying to angle it, often you will get a convex table.

The window can be lengthened some without significant effect.
