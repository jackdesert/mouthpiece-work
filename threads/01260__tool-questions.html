I do not know about any problems with the Dremmel.  I have an old Weller
Rotary tool that I have repaired several times.  I have a generic flex
shaft attachment I got for it from Micro-Mark.  I also have a variable
speed foot petal by Dremmel which seems real solid.  I rigged up my own
post to hold the motor.  There is a pic in the photos section.

This is not an essential tool to get started with.  But when you get into
interior modifications, it is great to start with the rotary tool then
finish off with hand filing.
