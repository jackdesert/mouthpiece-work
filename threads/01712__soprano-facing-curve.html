Hi There,
Would someone please be so kind as to give me measurements so that I 
can rescue a Selmer Soprano S80 D mpc which I have almost bodged by 
going too heavy on the table, making the tip opening too thin. I 
guess that I can say that I learn more from my mistakes, and the 
bigger the mistake the more that I learn!!!
The biggest lesson to learn I think is patience and not to rush jobs.

I think I want something like a 18mm Facing (0.0015) and a tip 
opening of about 1.5mm
I will be using the feelers .0015 .010 .024 .034 .050 as listed in 
the Eric Brand Manual.

Many thanks
I can't begin to thank you so much for the value of this Group, I am 
so pleased that I have discovered it!.

FROM: dolgishev (dolgishev)
SUBJECT: Re: soprano facing curve
tip opening = 1.5 mm
feeler(in)   from tip(mm)
   .0015         18.0
   .010          12.6
   .024           7.7
   .034           5.2
   .050           1.7
FROM: kwbradbury (Keith Bradbury)
SUBJECT: Re: soprano facing curve
dolgishev's numbers look good if you measure the 1.50 mm tip at the very
tip of the mouthpiece.  If you measure the tip inside the tip rail, and the
rail is .75mm thick, then the curve given is for a 1.40 mm tip opening.

Peter, Is this OK or would you like numbers for 1.50 mm as measured inside
the tip rail?
